$$ \left(x_i^{(k+1)}, y_i^{(k+1)}\right) = \left(\frac{x_i^{(k)}+x_{i+1}^{(k)}}{2}, \frac{y_i^{(k)}+y_{i+1}^{(k)}}{2}\right) $$

$$ \left(x_n^{(k+1)}, y_n^{(k+1)}\right) = \left(\frac{x_n^{(k)}+x_{1}^{(k)}}{2}, \frac{y_n^{(k)}+y_{1}^{(k)}}{2}\right) $$

  2. Try with $n=50$. Plot the coordinates in the last iteration as a line plot using matplotlib, with the last point joining back to the first one. Your graph should converge into some nice shape as number of iteration increases. What does the shape look like in general?

Q3¶

(4 marks)¶

The 2nd derivatives of a multivariable scalar function $f(x_1, x_2, ..., x_n)$ can be packed into a matrix form called Hessian matrix $\mathcal{H}$, which looks like:

$$ \mathcal{H}(x_1, x_2,...,x_n) = \begin{bmatrix} \frac{d^2f}{dx_1^2} & \frac{d^2f}{dx_1dx_2} & \cdots & \frac{d^2f}{dx_1dx_n} \\ \frac{d^2f}{dx_2dx_1} & \frac{d^2f}{dx_2^2} & \cdots & \frac{d^2f}{dx_2dx_n} \\ \vdots & \vdots & \ddots & \vdots \\ \frac{d^2f}{dx_ndx_1} & \frac{d^2f}{dx_ndx_2} & \cdots & \frac{d^2f}{dx_n^2} \\ \end{bmatrix} $$

Write a function H() that can compute the Hessian matrix of the function

$$f(x_1, x_2, ..., x_n) = x_1^3x_2^3 ... x_n^3$$

such that the result is returned in form of a NumPy array. The function is required to be able to determine $n$ automatically by how many inputs the user is giving to H(), e.g. if the user calls H(x1,x2), the output should be a 2x2 matrix; if the user calls H(x1,x2,x3,x4), the output should be a 4x4 matrix.

Bonus: Construct the matrix entirely with NumPy broadcasting (i.e. no for loop, while loop or if-else clause in the whole program).
