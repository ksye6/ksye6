Deconvolute by FFT/iFFT¶

The spirit of FFT on a sequence of length $M=2^n$ is that we only need to compute once the factors $\exp{\left(\frac{2\pi ik}{M}\right)}$ for $k=0$ to $\frac{M}{2}$, and keep reusing them at each subsequent level. This is possible because of the properties:

$$ \begin{align*} \exp{\left(\frac{2\pi i k}{\frac{M}{2}}\right)} &= \exp{\left(\frac{2\pi i(2k)}{M}\right)} \\[1.1em] \exp{\left(\frac{2\pi i\left(k+\frac{M}{2}\right)}{M}\right)} &= \exp{\left(\frac{2\pi ik}{M}+i\pi\right)} = - \exp{\left(\frac{2\pi ik}{M}\right)} \end{align*} $$
In [8]:
# only works for sequence of length M=2^n

def FFT(seq, inverse=False):
    
    M = len(seq)
    if inverse:
        W = np.exp(-1j*2*np.pi/M*np.arange(M//2))
    else:
        W = np.exp(1j*2*np.pi/M*np.arange(M//2))
    
    def FFT_recursive(seq):
        L = len(seq)
        while L > 2:
            
            H0 = FFT_recursive(seq[::2])
            H1 = FFT_recursive(seq[1::2])
            
            return np.concatenate((H0+W[::M//L]*H1, H0-W[::M//L]*H1))

        return np.array([seq[0]+seq[1], seq[0]-seq[1]])
    
    return FFT_recursive(seq)/M if inverse else FFT_recursive(seq)

Note: The given boundary conditions are NOT periodic because the functions at $y=0$ and $y=2$ are not the same.

In [14]:
Delta = 0.01
x = np.arange(0,1+Delta,Delta)    
y = np.arange(0,2+Delta,Delta)

J = len(x)-1
L = len(y)-1

j = np.arange(1,J)    # not include values at boundary
l = np.arange(1,L)

m = np.arange(1,J)
n = np.arange(1,L)

The boundary condition $\phi(x,y=2)=\cos\left(\frac{\pi x}{2}\right)$ is inhomogeneous. So the first thing to do is to modify the source from $\rho_{j,L-1}=0$ to $\rho_{j,L-1}=0-\frac{f_j}{\Delta^2}$ (while anywhere else remains $0$).

In [15]:
# compute rho_jl. Modify the edge J-1 to match the boundary condition.
f_j = np.cos(np.pi/2*j*Delta)
rho = np.zeros((J-1, L-1))
rho[:,-1] = -f_j

# perform DST on rho
rho_mn = np.einsum("jl, jm, ln -> mn", rho, np.sin(np.pi*np.outer(j, m)/J), np.sin(np.pi*np.outer(l, n)/L))

Then solve the Laplace's equation by DST.

$$\phi_{j+1,l}+\phi_{j-1,l}+\phi_{j,l+1}+\phi_{j,l-1}-4\phi_{j,l} = \rho_{j,l}\Delta^2$$$$\Downarrow$$$$\tilde{\phi}_{mn} = \frac{\tilde{\rho}_{mn}\Delta^2}{2\left(\cos\left(\frac{\pi m}{J}\right)+\cos\left(\frac{\pi n}{L}\right)-2\right)}$$
In [16]:
# compute phi_mn
M,N = np.meshgrid(m, n, indexing='ij')
denom = 2*(np.cos(np.pi*M/J) + np.cos(np.pi*N/L)-2)

phi_mn = np.divide(rho_mn, denom, out=np.zeros_like(rho_mn), where=(denom != 0))

Finally inverse DST the results.
